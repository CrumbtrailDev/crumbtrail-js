## Default Permission

Default permissions for the crumbtrail plugin

#### This default permission set includes the following:

- `allow-start-session`
- `allow-end-session`
- `allow-append-events`
- `allow-write-blob`
- `allow-flag-bug`
- `allow-write-bug-voice`
